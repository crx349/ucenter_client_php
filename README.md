uc_client根据官方版本改版，支持gbk和utf8格式的PHP程序接入。


## 主要特性

* 官方版本修改，兼容性好
   
* 支持gbk和utf8格式的php程序接入ucenter
  
* 根据对接程序自动判断gbk或utf8格式，防中文乱码
  

## 安装使用

git clone https://gitee.com/crx349/ucenter_client_php uc_client
